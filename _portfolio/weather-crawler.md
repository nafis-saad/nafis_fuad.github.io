## Overview

An automated data collection pipeline that crawls and processes meteorological data from three major weather data sources: **Bangladesh Meteorological Department (BMD)**, **European Centre for Medium-Range Weather Forecasts (ECMWF)**, and the **Global Forecast System (GFS)**.

## Features

- Automated scheduled data retrieval
- Multi-source data harmonization
- Integration-ready outputs for geospatial and transportation models
- Error handling and retry logic for robust data collection

## Tools & Technologies

- **Python** — Web scraping and automation
- **Jupyter Notebook** — Pipeline documentation
- **requests / BeautifulSoup** — Web crawling
- **pandas** — Data processing
