## Overview

This project applies machine learning classification algorithms to predict **landslide susceptibility** across a study region. By combining terrain features, soil data, rainfall patterns, and historical landslide records, the model generates susceptibility maps that can inform land use planning and disaster risk reduction.

## Methodology

- Feature extraction from DEM (Digital Elevation Model), slope, aspect, curvature
- Integration of rainfall and soil data
- ML classification (Random Forest, SVM, etc.)
- Susceptibility map generation and validation

## Tools & Technologies

- **Python** — ML modeling and analysis
- **Jupyter Notebook** — Reproducible analysis
- **GIS** — Spatial data processing
- **scikit-learn** — Machine learning pipeline
