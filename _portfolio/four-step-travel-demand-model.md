## Overview

A comprehensive implementation of the **four-step travel demand modeling** framework for the Sacramento Region. This is one of the most widely used transportation planning tools, and this project brings it to life using modern GIS and Python tooling.

## What is a Four-Step Model?

The four-step model is the standard framework in transportation planning:

1. **Trip Generation** — How many trips start and end in each zone?
2. **Trip Distribution** — Where do those trips go?
3. **Mode Choice** — How do people travel (car, transit, walking)?
4. **Traffic Assignment** — Which routes do they take?

## Tools & Technologies

- **TransCAD** — Transportation planning software for network modeling
- **ArcGIS Pro & arcpy** — Geospatial processing and visualization
- **Python** — Data processing, automation, and analysis
- **Jupyter Notebook** — Analysis and documentation

## Key Features

- Full four-step model pipeline for Sacramento Region
- Integration of socioeconomic and land use data
- GIS-based visualization of travel patterns
- Reproducible Python scripts for each modeling step
